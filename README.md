📊 Instagram: Improving Engagement Quality & Retention
📌 Overview
This project analyzes Instagram user behavior to understand the difference between meaningful engagement vs passive usage.
The goal is to improve user retention and engagement quality using data-driven insights.

📂 Dataset
1.5M+ users
60+ features
Includes:
User activity (sessions, time spent, interactions)
Engagement metrics (likes, comments, DMs)
Lifestyle factors (stress, sleep, etc.)


⚙️ Tech Stack
Python
Pandas, NumPy, SciPy
Matplotlib, Seaborn
Tableau


📈 Key Work
Performed end-to-end EDA on large-scale user dataset
Created Engagement Quality Efficiency (EQE) metric
Conducted segmentation & behavioral analysis
Identified churn-risk and passive usage patterns
🔍 Key Insights
~39% users show high session frequency but low engagement quality
~23% users identified as passive scrollers
~7% users fall into high churn-risk segment
Strong negative correlation between stress and engagement quality


📊 Output
Data analysis in Python (Jupyter Notebook)
Visualizations for trends & patterns
Tableau dashboard for interactive insights


🚀 Business Impact
Identified opportunities to improve:
User retention
Engagement quality
Product experience
